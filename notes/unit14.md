# 重载运算与类型转换
## 14.1 基本概念
如果运算符函数是成员函数，它的左侧运算对象绑定到隐式的this指针，所以少一个参数

优先级和结合律保持一致
### 可以直接调用一个重载的运算符函数
```c++
// 非成员
data1 + data2; // ==
operator+(data1, data2);
// 成员
data1 += data2;
data1.operator+=(data2); //==
```
### 选择作为成员或者非成员
* `=, [], (), ->`必须是成员
* 复合赋值一般应该是成员
* 递增，递减，解引用应该是成员
* 算术，相等性，关系，位应该是非成员

```c++
string s = "world";
string t = s + "!";
string u = "hi" + s; // 如果+是string的成员，则等于“hi”.operator+("!"),显然const char*没有成员函数
```
