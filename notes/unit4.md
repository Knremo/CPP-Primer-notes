### 左值和右值
当一个对象被用作右值时，用的是对象的值；当对象被用作左值时，用的是对象的身份（在内存中的位置）

在需要右值的地方可以用左值代替

`decltype`作用于左值得到一个引用类型，假定`int *p`，解引用生成左值，所以`decltype(*p)`的结果是`int&`

取地址符生成右值，所以`decltype(&p)`的结果是`int**`

# 4.4 赋值运算符
左侧运算对象必须是一个可修改的左值

结果是左侧运算对象，所以

```c++
if (i = 0) cout << 1 << endl; //不会有0
if (i = 1) cout << 2 << endl; //只有1
```
